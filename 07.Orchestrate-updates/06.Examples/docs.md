This section provides working examples of Mender Orchestrator performing
System-level updates across multiple Components.

* [Mock components](01.Mock-components/docs.md) - a self-contained walkthrough
  using the demo mock environment, with a Linux System Device and two simulated
  RTOS Components. No hardware required.
* [Update an ESP32-S3 with a Raspberry Pi](02.Update-an-ESP32-S3-with-a-Raspberry-Pi/docs.md) -
  a real-hardware example where a Raspberry Pi 5 updates an ESP32-S3
  microcontroller that does not run Mender.
